# Sleepy Productions

> Building local-first, privacy-respecting software — to make the world a little freer, one app at a time.

Independent software studio (Sleepy Productions LLC). The work runs the gamut: native desktop apps, developer tooling, AI agents, and bespoke websites for local businesses. A few guiding rules:

- **Local-first.** Your data stays on your machine. No accounts required, no uploads, no telemetry by default.
- **Free & honest.** No ads, no dark patterns, no tracking. What it says on the tin is what it does.
- **Built to last.** Strong baseline first, then polish — software that holds up.

### What I work in

`Rust` · `Tauri` · `TypeScript` · `Python` · `Astro` + `Tailwind` · `Cloudflare Workers / Pages` · `Luau`
